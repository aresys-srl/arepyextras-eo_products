Installation
============

Requirements
------------
* Python 3.9 or higher.


Install
-------
1. Install the Arepyextras EO Products package and its dependencies using ``pip`` tool:

.. code-block:: bash

    $ pip install arepyextras-eo_products

2. Install the Arepyextras EO Products package and its dependencies using ``conda`` tool:

.. code-block:: bash

    $ conda install arepyextras-eo_products
